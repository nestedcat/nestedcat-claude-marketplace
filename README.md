# nestedcat-claude-marketplace

A personal Claude Code plugin marketplace by nestedcat.

## Install

```bash
/plugins marketplace add nestedcat/nestedcat-claude-marketplace
```

## Available plugins

| Plugin | Description |
|--------|-------------|
| [plugin-audit](https://github.com/nestedcat/plugin-audit) | Capability inventory & security audit tool for Claude Code |

## Usage

After adding the marketplace, install plugins with:

```bash
/plugins install plugin-audit@nestedcat-claude-marketplace
```
